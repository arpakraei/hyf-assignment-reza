# Portfolio Page

This is a simple personal portfolio webpage created using HTML, CSS, and JavaScript.  
The page introduces me and includes a small interactive feature that allows the user to change the background color.

## Features

- Simple personal introduction
- List of interests and hobbies
- Interactive button that changes the background color randomly
- Clean and responsive layout

## Technologies Used

- **HTML** – page structure
- **CSS** – styling and layout
- **JavaScript** – interactivity and random color generation

## How It Works

When the user clicks the "Change Background Color" button, a JavaScript function selects a random color from a predefined list and applies it to the page background.
